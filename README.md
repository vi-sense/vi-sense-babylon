# Vi-sense Frontend

## Usage

### Installation

```
$ npm install 
$ npm run start
```

### Building

```
$ npm run build 
```

# Some notes and conventions for programming

```
export class App {

    #someVariableName;

    constructor() { ... }
}
```

`#someVariableName` is a private member

all static (images, models, ...) files belong to the `/public` folder
